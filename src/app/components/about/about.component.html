About View

Shortener Url generator logic:

There is a lot of ways to shorten Url, but I choose way, in which your url address convert to short url address,by using my own converter(Also you can create it by yourself).

1) It gets 5 last symbols of original url.

2) Than places random values between this symbols.

3) Generate unique url address with 10-11 characters.

Shortener Url redirection logic:

Here I will describe the most simple way to redirect from generated url to original url:

1) I save original and shortened urls like a pair in database.

2) Than after following shortened link, user will be redirected on original link (client makes request on my REST API and get from DB original link, by the link, which was entered by user).

3) I used redirection action from ASP .NET MVC Controller.
